# matsu.num.Number
matsu.num.Number は Java 言語向けの数値計算における, 整数論に関するライブラリである.

現在のリリースバージョンは 0.2.0 であり, Java 17 に準拠する.
このバージョンにおいて, 次がサポートされている.

- 整数の乗算に関するユーティリティ
- 剰余演算
- 素数判定
