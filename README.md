# matsu.num.Number
matsu.num.Number は Java 言語向けの数値計算における, 整数論に関するライブラリである.

現在のリリースバージョンは 1.0.1 であり, Java 17 に準拠する.
このバージョンにおいて, 次がサポートされている.

- 乗算に関するユーティリティ
- GCD (最大公約数) の計算
- 剰余演算
- 素数関連
  - 素数判定
  - 素因数分解
